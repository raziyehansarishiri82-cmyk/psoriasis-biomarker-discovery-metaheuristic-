Overview

This repository accompanies the published article introducing an integrated computational framework for biomarker discovery in psoriasis.

The proposed framework combines:

Gene expression analysis
Evolutionary feature selection algorithms
Metaheuristic optimization
Machine learning classifiers

to identify highly informative diagnostic genes associated with psoriasis.
Main Contributions
Integration of seven GEO datasets
Preprocessing and normalization of gene expression profiles
Feature selection using:
Harris Hawks Optimization (HHO)
Genetic Algorithm (GA)
Particle Swarm Optimization (PSO)
Classification using:
XGBoost
Random Forest
SVM
AdaBoost
Decision Tree
Identification of 35 diagnostic biomarker genes
Best performance achieved by the HHO–XGBoost framework (F1-score = 99.31%)
